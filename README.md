# Weather App

这是一个简单的天气应用，它可以显示当前的天气信息和未来几天的天气预报。

## 功能

- 显示当前的日期和时间
- 显示当前的天气信息（如温度，湿度等）
- 显示未来几天的天气预报
- 提供搜索功能，可以查看全球任何城市的天气信息

## 技术栈

- HTML
- CSS
- JavaScript
- Bootstrap

## 如何运行

1. 克隆或下载此仓库
2. 打开[`index.html`]文件在你的浏览器中
3. 你可以看到应用的主界面，并可以开始使用
